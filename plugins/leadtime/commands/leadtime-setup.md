Start the guided Leadtime workspace setup with the **`leadtime-setup`** skill.

Begin with **Phase 0**: connect through Leadtime MCP, inspect the current workspace state (signed-in person, workspace name, existing users, organizations, and projects), and greet the user briefly. Then show a compact roadmap and ask the **Phase 1 interview questions** before starting the first step.

Follow the `leadtime-setup` `SKILL.md` strictly and work through `assets/onboarding-steps.json` in order. Handle one step at a time and close each step with the video/read-more learning block.
